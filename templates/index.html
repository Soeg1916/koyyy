Social Media Downloader Bot

About the Bot

This Telegram bot allows you to download videos and images from various social media platforms including:

  • TikTok
  • Instagram
  • YouTube Shorts
  • Pinterest (images and videos)

How to Use

To use the bot, follow these steps:

  1. 1. Start a chat with the bot on Telegram
  2. 2. Send a URL to a social media post you want to download
  3. 3. The bot will download the media and send it back to you
  4. 4. You can extract audio from videos or save media for later use

Bot Commands

Command Description
/start Begin interaction with the bot
/help View available commands and usage instructions
/list View all your saved media
/my [name] Retrieve a specific saved media by name
/cancel Cancel current operation

Start using the bot by messaging @YourBotUsername on Telegram
